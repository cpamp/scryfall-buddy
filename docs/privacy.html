Privacy Policy for Scryfall Buddy

Effective date: April 9th, 2026

Scryfall Buddy provides enhancements to the Scryfall website, including allowing users to save and manage search terms.

Summary

This extension stores only the search terms that users explicitly choose to save, along with any minimal settings needed for the extension to work.

This extension uses the browser's extension storage APIs, including sync storage when available, so that saved data can persist and sync across the user's signed-in browser instances.

This extension does not send saved search terms to the developer's own servers.

This extension does not sell, rent, or share user data with advertisers, data brokers, or other third parties.

What data this extension collects

Scryfall Buddy may collect and store:

  • Search terms that the user explicitly chooses to save
  • Basic extension preferences or settings needed for functionality

How the data is used

The data is used only to provide the extension's user-facing functionality, including:

  • Saving search terms selected by the user
  • Displaying saved search terms back to the user
  • Allowing the user to edit or delete saved search terms
  • Syncing saved search terms and settings across the user's browsers or devices when browser sync is enabled

Where the data is stored

The data is stored using the browser's extension storage APIs, including sync storage when available.

The developer does not transmit this data to their own servers.

Data sharing

Scryfall Buddy does not sell, rent, or share saved search terms with third parties.

The only storage or transfer involved is the browser-provided sync functionality used to keep the user's extension data available across signed-in browser instances.

Data retention and deletion

Saved search terms and related settings remain stored until the user:

  • Deletes them within the extension
  • Clears the extension's stored data
  • Uninstalls the extension

Retention may also be affected by the browser's own sync settings and account behavior.

Security

Scryfall Buddy relies on the browser's built-in extension storage and sync mechanisms to store user data.

Scryfall Buddy does not operate its own server for this data.

Limited Use Disclosure

Scryfall Buddy limits its use of user data to the practices described in this policy.

Scryfall Buddy uses user data only to provide and improve the extension's user-facing functionality of saving, syncing, displaying, editing, and deleting search terms selected by the user.

Scryfall Buddy does not collect or use web browsing activity, except as required to provide its user-facing features on Scryfall pages where the extension runs.

Scryfall Buddy does not use user data for advertising, including personalized advertising.

Scryfall Buddy does not sell user data.

Scryfall Buddy does not transfer user data to the developer's own servers or to third parties, other than the browser's built-in sync functionality used to keep extension data available across the user's signed-in browser instances.

Scryfall Buddy does not transfer or sell user data to advertisers, advertising platforms, data brokers, information resellers, or for credit-worthiness or lending purposes.

The developer of Scryfall Buddy does not access or allow humans to read users' saved search terms, except if required to comply with applicable law or to protect against abuse, fraud, or security threats.

All other transfers, uses, or sale of user data are prohibited.

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes to this policy

This privacy policy may be updated from time to time. Any changes will be reflected by updating the effective date above.
