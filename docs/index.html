Scryfall Buddy

Scryfall Buddy is a lightweight browser extension for Scryfall that adds smarter query suggestions, a visual search builder, and saved favorites backed by your browser's extension storage.

What it does

Scryfall Buddy adds operator suggestions, otag and color suggestions, and a visual builder for composing searches on Scryfall pages.
